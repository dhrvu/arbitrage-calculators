#### Calculators


Mathematical tools for Odds analysis.


## Calculators & Formulae



### Arbitrage Calculator 

Formula :

Total Implied Probability:  
`T = Σ(1/odd_i)`  
Profit Calculation:  
`Profit = (Stake / T) - Stake`



### Kelly Criterion

Formula :

Optimal Stake:  
`f = (bp - q) / b`  
Where:  
`b = decimal odds - 1`  
`p = win probability`  
`q = 1 - p`



### Hedging Calculator

Formula :

Hedge Stake:  
`H = (S₀ × O₀) / O₁`  
Guaranteed Profit:  
`P = (S₀ × O₀) - H`



### Dutching Calculator

Formula :

Stake per Outcome:  
`S_i = (Total Stake / odd_i) / T`  
Where `T = Σ(1/odd_i)`

### Bonus Bet Calculator

Formula  : 

Total Return:  
`R = (Bonus × (B_odds - 1)) + (Normal × (N_odds - 1))`
